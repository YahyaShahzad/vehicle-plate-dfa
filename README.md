# Pakistan Vehicle Plate Validator (DFA Based)

A web-based **Pakistan Vehicle Plate Validation System** built using 
**Deterministic Finite Automata (DFA)** and **Flask**.

This project validates vehicle number plates for **all provinces of Pakistan**
according to official formats for **2-wheelers and 4-wheelers**.

---

##  Features

- DFA-based validation (Theory of Automata implementation)
- Supports all Pakistani provinces
- Separate formats for 2-wheelers & 4-wheelers
- Real-time plate preview
- Validation history (last 20 records)
- Interactive charts using Chart.js
- Responsive & professional UI

---

##  Theory Used

- Deterministic Finite Automata (DFA)
- Formal language pattern recognition
- State transitions based on character types

---

##  Supported Provinces

- Punjab 
- Sindh 
- Balochistan 
- Khyber Pakhtunkhwa 
- Gilgit Baltistan 
- Azad Jammu & Kashmir 
- Islamabad Capital Territory 

---

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Charts:** Chart.js
- **Theory:** DFA (Automata Theory)
